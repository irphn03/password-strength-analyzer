# Password Strength Analyzer

A simple Python-based mini project to check the strength of a password using rule-based validation.

## Features
- Checks password length
- Detects uppercase and lowercase letters
- Detects numbers
- Detects special characters
- Classifies password as Weak, Medium, or Strong

## Technologies Used
- Python
- Regular Expressions (re module)

## How to Run
1. Run the file using: python password_checker.py
2. Enter a password when prompted
3. The program will display password strength
